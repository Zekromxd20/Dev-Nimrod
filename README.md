# 🚀 Dev Nimrod: Agentic Code Auditor

**Dev Nimrod** is an Agentic AI system built for the 2026 Hackathon. It goes beyond simple chat—it audits, explains, and refactors code with a focus on developer trust and traceability.

## ✨ Features
- **Intelligent Refactoring:** Powered by Gemini 3 Flash for near-instant $O(n)$ optimization logic.
- **Diff View:** Side-by-side visual comparison of changes (Git-style).
- **Reasoning Engine:** Don't just get new code; understand *why* the AI changed it.
- **Agentic Logic:** Identifies complexity bottlenecks (e.g., $O(n^2) \rightarrow O(n)$).

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **AI Engine:** Google Gemini 3 Flash
- **Deployment:** Streamlit Cloud

## 🚀 Quick Start
1. Clone this repo.
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `GOOGLE_API_KEY` to your environment or Streamlit secrets.
4. Run: `streamlit run app.py`
